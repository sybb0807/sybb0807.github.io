新手快速搭建个人博客并部署到远端教程(hexo + github)
发表于 2020-05-02| 更新于 2020-05-03|hexo
阅读量:

作为一个程序员， 有一个自己的博客， 在每一次写下一篇博客的同时，也记录着自己的成长历程。一个好的博客在求职的过程中也会为自己大大加分。

搭建博客的方式可是太多了，之所以选择hexo+github的方式，一方面是因为github本身就是程序员们离不开的一个东西，直接用github会比较方便，另一方面就是不用自己买服务器啦，省去了自己服务器钱以及自己维护的成本，还有就是会稳定吧。

本人也是第一次用hexo搭建博客，就把搭建过程记录下来，感觉还挺简单的，照着做下来，20分钟不到就能搭起来了。

准备资源

  1. 准备node.js

    查看本机有没有node.js

    如果出现版本那就是有，提示没有node命令的话，那就是没有，安装吧。

    安装node.js直接到node.js官网下载就好。

    安装后还是执行node -v 检查是否安装成功

  1. 安装包管理

    按道理node.js安装成功后就会有npm了，查看本机是否有npm：

    道理和node一样，提示没有npm命令那就是没有装成功。

    但是，npm因为是国外的服务所以会有速度慢的问题，所以很多人也会用cnpm去装包，网上很多安装cnpm相关资料，不再赘述。但是again，cnpm有时会存在丢包的问题，所以本人选择用yarn，传送门：yarn安装和使用。

    这三种安装方式都能装，选哪个还是看个人习惯。

部署到远端

  1. 创建仓库

    登录自己的github账号，创建新的仓库，并把仓库名命名为 自己的github账号+ ‘github.io’的格式，如‘sybb0807.github.io’。

    仓库名一定要与自己的github账号一致，不然最后直接访问仓库名会是404

    直接在浏览器中访问 https://你的github账号.github.io/，如：https://sybb0807.github.io/ 即可访问你的博客啦。
